## ✨ Features

### Web Dashboard
- 📋 Add, view, and delete students
- ➕ Record subject marks with automatic % calculation
- 📊 Grade breakdown per student with letter grades (A / B / C / F)
- 📈 Charts — grade distribution bar chart & pass/fail doughnut chart
- 🔍 Live search across all students
- ⬇️ CSV export of all results
- 💾 LocalStorage persistence — data saved across sessions

### Java Backend (CLI + Database)
- 🗃️ SQLite database via JDBC — persistent storage
- 🧱 Clean MVC architecture — Model, Service, Database layers
- 👤 Full CRUD for students and grades
- 📄 Formatted reports printed to terminal
- ✅ Input validation at service layer
- 📊 Aggregated SQL queries — class average, pass/fail counts
